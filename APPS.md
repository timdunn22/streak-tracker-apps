## Stack

Vite 7 + React 19 + TypeScript + Tailwind CSS 4 + vite-plugin-pwa

## Features

- Live streak timer with animated counter
- Recovery timeline with body/brain healing phases
- Daily check-in with rotating encouragement
- Journal with mood tracking and trigger logging
- Breathing exercise (4-7-8 technique)
- Milestone celebrations with confetti and chime
- Streak freeze (2 per recharge cycle)
- Badge and 10-level progression system
- Money saved calculator (6 apps)
- Share card generator (1080x1350, optimized for social)
- Streak history sparkline with trend analysis
- Full offline support as installable PWA
- Swipe navigation between tabs
- Keyboard accessible with ARIA live regions
- Works on all devices (320px to desktop)
